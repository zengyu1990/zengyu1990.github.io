Goodbye Typecho and Hello Hexo

发表于 2016-05-01   |     |  

今天是2016年5月1日。我开始写第一篇在Hexo的文章，从开始准备迁移到完全搭建好一共耗时一个多月吧，其实就是自己一个人瞎折腾，很多时候搭建到一半遇到问题又推倒重来，中间选主题也是各种纠结。

Maupassant是之前在Typecho选的，简单到极致，配上Zen这个title让我觉得这个东西会陪我到永远，在Hexo也有人移植了这个主题，可惜字体跟细节还是没有完善，最终我还是选了Hexo最多人star的主题，Next。

再见啦陪我一年半的Typecho；你好Hexo，你会陪我多久呢？

Typecho

Typecho是我自己搭建的第一个博客，服务器是用Sae的，域名是自己的Godaddy买的，之前利用职务之便在Sae开发者认证中将自己的账号认证为高级开发者，那样每个月免费10000云豆（每天上万uv都不需要付费）。

2014年年底开始要求自己每个月都要在自己的博客写一篇文章，现在回首看看也算是勉强达标吧，但是后期写的文章明显就是为了交差而写的，没有去构思，然后文章随便写完也就丢上去了。

如果不是因为Sae停止了免费云豆的赠送，我猜我之后也就是那样凑合着用Typecho，凑合着去完成每个月一篇的任务吧。不过Sae要开始收费了，而且可预见的未来，每天大概五毛钱的费用，还不知道以后会不会涨价。

于是我开始搜寻新的地方来放我的博客，小彭彭建议我用Hexo。然后我发现原来这个东西之前就有人介绍过我用，只是当时蛮喜欢Typecho极简的理念，而且Typecho很方便与Sae捆绑在一起，然后没有对Hexo作进一步的了解。

我在网上继续搜寻了下相关的个人博客解决方案。有很多，例如：FarBox、Jekyll、Octopress、ghost、marboo、Hexo、Medium、Logdown、prose.io。经过筛选后好像没有比Hexo更适合的了。

Hexo

Github Pages + Hexo + Next = Sae + Typecho + Maupassant

简单得说就是之前在Sae上曾经以为会永久免费的个人博客现在要转移到Github上去了，虽然Github像Sae那样从免费变成收费的概率很小，但是最让我担心的是Github被墙，特别是看到自由门这种软件都放在Github Pages之后。

当年搭建Sae上的博客是用svn的，现在在Github上当然就得用git了，不过好在我现在也有macbook了，按照网上的教程加上自己三脚猫的技术基础，也算是可以磕磕碰碰得将博客搭建起来。

特别感谢作者在做Next这个主题的同时还做了一个专门的文档教程，教程的指引下我配置好了整个博客。虽然这个主题本身就是以极简出名的，但是我还是删掉了很多东西，同时费尽心思将一些细节进行修改，例如顶部Zen标题的大小，我是通过Chrome查看去定位css文件，然后再找到对应的字体大小设置，最终才改好的（对于我这种非coder，这已经是很困难的了）。

七牛

其实我以后还打算在博客里新增关于页面（About），同时也在考虑新增一个图墙页面，那么我的图片就需要放在一个图床里。七牛以前用过，好像也是可以免费用很长一段时间，但是现在好像又特别怕它像Sae那样走上收费的路，那样我的博客岂不是要在某个服务停止之后瘫痪掉？

GoDaddy

很久很久以前我就在想，什么时候我可以拥有一个属于自己的顶级域名，我要.com的，那样感觉与其他所有知名的网站平起平坐了，像什么google.com、facebook.com。而且在网易Lofter写过一篇文章，当时我写了一个上午。在点发布的时候，系统提示我有敏感词，不能发送！我后来根据二分法定位到了敏感词是“轮子功”，卧槽我明明已经做了敏感词的规避结果还是要被和谐。那次之后我就慢慢淡了再Lofter上写东西的热情了，转而期盼有自己的博客。听说国内万网可以买域名，但是受到国内的监管，GoDaddy上买域名也支持支付宝支付了。果断放弃万网买的打算，转而开始在GoDaddy上试域名，最终在某个早上给我选到了z1990.com这个域名，很短也很有寓意而且一年只要69块人民币，果断抛弃了之前申请的zen1990.com。

DNSPod

在Godday买了域名后，还要在国内备案。当初公司在阿里云备案网站的时候，搭了个顺风车，将z1990.com这个域名也搞下来了。至于具体要怎么将域名与博客绑定在一起就要靠DNSpod了，其实很多东西走对了就水到渠成得搞定了，像DNSpod的话按照网上的教程很容易就将Github Pages于其绑定在一起了，那么我的博客就不再是没有故事的github.io的二级域名了，而是国际顶级短域名：www.z1990.com!

Markdown

这个传说中被程序员誉为文档神器的东西，可以让人专注写作的软件。也在我选择了Hexo后不得不进一步去使用了，在网上找了教程就开始码这篇文章了，虽然截止到现在也只是用了标题语法，但是随着以后更认真得去写一些东西的话，希望可以像归档里的那句话，生命里的每一个点都会最终连成线。

后记：

朋友圈看到过一句话：每个人都应该热爱一件无关重要的小事，不靠它获利，也考它获不了什么利，但你偏偏对其如痴如醉不能自拔，捣鼓这个博客就是属于我的那件小事！
